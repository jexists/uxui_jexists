#### Mixin 하단 인디케이터 형태

선택자일경우 변수를 그대로 사용X =>**interpolation**(보간/대처)사용: 형태 그대로 대입시키도록 처리

```scss
#{$선택자}
```

```scss
@mixin indicator_01($ind_class, $myW){
	//하단 인디케이터 형태, focus처리되면 해당 버튼이 길게 늘어나는 형태
	//OPT_01: 인디케이터 버튼 하나의 크기 $myW로 설정
	//OPT_02: class선택자의 이름 설정 $ind_class
	//[기본구조] $ind_clss > ul > li > a 순서
#{$ind_class}{ $w:$myW; $f:100%;
 	position: absolute; left: 50%; bottom:$w; transform:translateX(-50%);
 	margin: auto; width:auto; min-width: 135px; 
 	height: $w; padding: 5px; 
 	border-radius: $w; background-color: $wt;
 >ul{width: $f; height: $f;
  >li{float:left; width: auto; height: $w; 
  	margin-left: 5px;  
  &:first-child{margin-left: 0;}
	a{display: block; width: $w; height: $f; border-radius: $w;
		background-color: $gy; transition: all 300ms ease;
	 &:hover{background-color: lighten($pmC,30%);}
	 &:focus{background-color: $pmC; width: ($w * 3); border-radius: $w;
	 	outline: none;}}}}}
}// mixin indicatory_01
// 불러오기 =========================================

```

#### 가로형 슬라이드 색깔 자동으로 바꾸기 - 반복문

[js] for(최초의 임의 변수; 반복횟수; 수행하기 위한 연산){}
[js] for(let i=0; i<$slideN; i++){}
[scss]  @for$변수 from 시작값 through/to 끝값 "to <, through <="

```scss
@for $i from 0 through $slideN {원하는 값}
```

[예제]

```scss
//가로형 슬라이드 배너
.slide_form{ $slideN:4;
	width: 100%; height: 100%;
	.guide{width: (100%*$slideN); height: 100%; background-color: $spC;
		>div{float: left; width: (100%/$slideN); height: 100%; 
			&:nth-of-type(1){background-color: $wt;}
			&:nth-of-type(2){background-color: darken($wt,20%);}
			&:nth-of-type(3){background-color: darken($wt,40%);}
			&:nth-of-type(4){background-color: darken($wt,60%);}}}}
// 반복문 사용 =========
.slide_form{ $slideN:4;
	width: 100%; height: 100%;
	.guide{width: (100%*$slideN); height: 100%; background-color: $spC;
		>div{float: left; width: (100%/$slideN); height: 100%; 
			@for $i from 1 through $slideN {
				&:nth-of-type(#{$i}){background-color: darken($wt, $i * 20%);
			}//@for문		
        }}}}
```

#### 여러개의 이미지 넣기

background-image:url($url + 'slide_0#{$i}.jpg');
background-image:url($url + "slide_0" + $i + ".jpg");

```scss
//연속된 이름 이미지 넣기 ========================
$url : '../img/slide/';
.slide_form{ $slideN:4;
 overflow: hidden; width: 100%; height: 100%;
 .guide{width: (100%*$slideN); height: 100%; background-color: $spC;
	>div{float: left; width: (100%/$slideN); height: 100%; 
		background-repeat: no-repeat; background-position: 50% 50%;
		background-size: cover;
	 @for $i from 1 through $slideN {
		&:nth-of-type(#{$i}){
		// background-image:url($url + 'slide_0#{$i}.jpg');
		background-image:url($url + "slide_0" + $i + ".jpg");}//@for문
	}}}}
//연속되지 않은 이름 이미지 넣을때  ===================
$url : '../../img/slide/';
$myImg : ('slide_01.jpg','slide_02.jpg','slide_01.jpg','slide_01.jpg');
.slide_form{ $slideN:4;
	overflow: hidden; width: 100%; height: 100%;
	.guide{width: (100%*$slideN); height: 100%; background-color: $spC;
		>div{float: left; width: (100%/$slideN); height: 100%; 
				background-repeat: no-repeat; background-position: 50% 50%;
				background-size: cover;
			@for $i from 1 through $slideN {
				&:nth-of-type(#{$i}){
				background-image: url($url + nth($myImg, $i));}//@for문
		}}}}
//10장이상인 이미지 넣을때 if문 사용 ===============================
.slide_form{ $slideN:4;
	overflow: hidden; width: 100%; height: 100%;
	.guide{width: (100%*$slideN); height: 100%; background-color: $spC;
		>div{float: left; width: (100%/$slideN); height: 100%; 
				background-repeat: no-repeat; background-position: 50% 50%;
				background-size: cover;
			@for $i from 1 through $slideN {
				&:nth-of-type(#{$i}){
				$n : null;
				// background-image:url($url + "slide_0" + $i + ".jpg");
				@if ($i < 10 ){$n : '0' + $i; //01, 02 ... 09
				}@else{$n:$i;}
				background-image:url($url + 'slide_#{$n}.jpg');
				}//@for문
		}}}}
```
